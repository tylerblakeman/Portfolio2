Brew Crawler

Brew Crawler uses location data from Google Maps along with the Open Brewery Database and Firebase to plan a Brewery Crawl based on your location. Brew Crawler was a team project during the UW Extended Campus Code Boot Camp. This project was a collaboration with Andrew Mueller and Zach Hassler.

Trivia Game

The Trivia game was developed for the UW Extended Campus Code Boot Camp. The game was created with JavaScript and HTML. The game provides the user with 6 random questions and allows them 30 seconds to answer, if the user fails to answer they get an incorrect answer. At the end of the game the user is provided their score as a percentage.

Train Scheduler

Train scheduler was created using Firebase as storage for trains as they are scheduled by a user. As the page loads the saved trains are loaded from the Firebase database and the next arrival and minutes away are updated. The train scheduler was built for the UW Extended Campus Code Boot Camp.

Liri Node App

The Liri Node App is a text based node application that uses Open Movie Database, Bands in Town, and the Spotify API to provide responses to text based commands. This is the first project for the UW Code Boot Camp that I have provided extensive documentation on GitHub for.
